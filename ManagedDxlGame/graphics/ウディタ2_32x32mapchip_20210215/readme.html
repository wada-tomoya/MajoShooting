ウディタ２向け32×32マップチップ素材

●収録内容

ウディタ(WOLFRPGエディター)向けのマップチップ(タイルセット)素材です

ウディタ2同梱の16×16サイズのマップチップを32×32に作り直したもので、一部チップの種類や配置を変更したものの、ウディタでの利用では画像ファイルの上書きとゲーム設定の変更でほぼそのまま既存のマップを変換することができます

ウディタ以外のゲーム制作ソフトや自作プログラム、WEBサイト・コンテンツ制作にもご利用いただけます

オートタイル画像はウディタ用と他の用途でも使用できるよう構成パターンを展開したものも収録しています

2019年7月31日バージョンではダンジョン,雪が積もった街,破壊された街のチップは未対応のため、16×16サイズのものを単純に拡大したものを収録していますが、今後チップの種類や配置を変更する可能性がありますので、ご留意ください

今後の対応などについては進捗ある都度こちらに記載しております

ウディタ10周年おめでとうマップチップ素材 - Google スプレッドシート

「MapChip」フォルダ

ウディタ(WOLFRPGエディター)用のマップチップ画像ファイルです

「タイル設定ファイル」フォルダ

ウディタ(WOLFRPGエディター)用のタイル設定ファイルです

16×16サイズのものから一部チップの種類や配置を変更しているためそれに対応するものです

「オートタイル展開」フォルダ

ウディタ(WOLFRPGエディター)用のオートタイル画像を他用途で使用できるように、構成パターンを展開した画像ファイルです

アニメーションするものはアニメなしバージョンもあります
